# ALABTAL English AI App 🦸

تطبيق سلسلة الأبطال لحل أسئلة اللغة الإنجليزية بالذكاء الاصطناعي

## خطوات النشر على Vercel

### 1️⃣ رفع الكود على GitHub

1. اذهب إلى [github.com](https://github.com) وسجّل دخول
2. اضغط **New Repository**
3. اسم المشروع: `alabtal-english-app`
4. اضغط **Create Repository**
5. ارفع الملفات:
   - `package.json`
   - `public/index.html`
   - `src/index.js`
   - `src/App.js`

### 2️⃣ ربط GitHub بـ Vercel

1. اذهب إلى [vercel.com](https://vercel.com)
2. سجّل دخول بحساب Google
3. اضغط **Add New Project**
4. اختر مشروع `alabtal-english-app` من GitHub
5. اضغط **Deploy** ✅

### 3️⃣ الحصول على Gemini API Key (مجاناً)

1. اذهب إلى [aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)
2. سجّل بحساب Google
3. اضغط **Create API Key**
4. انسخ المفتاح
5. افتح التطبيق → لوحة المعلم → أدخل كلمة السر → الصق المفتاح

## معلومات التطبيق

- **كلمة سر المعلم:** hero2026
- **الذكاء الاصطناعي:** Google Gemini (مجاني)
- **الصفوف:** من الأول الابتدائي إلى الثالث الإعدادي
- **المادة:** اللغة الإنجليزية
